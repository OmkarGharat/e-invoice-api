Advanced Filtering

Available Filter Parameters:
Basic Filters:
  • ?status=Generated
  • ?supplyType=B2B
  • ?sellerState=29
  • ?buyerState=07
  • ?documentType=INV
Multiple Values (OR logic):
  • ?supplyTypes=B2B,EXPWP,SEZWP
  • ?statuses=Generated,Cancelled
Special Filters:
  • ?interstate=true
  • ?reverseCharge=false
  • ?totalValue=lt:1000
  • ?totalValue=gt:50000
  • ?search=INV/2024
Pagination & Sorting:
  • ?page=2&limit=20
  • ?sortBy=totalValue&sortOrder=desc
  • ?dateFrom=2024-01-01&dateTo=2024-12-31
  • ?minValue=1000&maxValue=10000
